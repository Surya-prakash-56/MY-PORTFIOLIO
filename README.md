# Portfolio Website

A personal portfolio website built with HTML, CSS, JavaScript, and a React/Vite demo app.

## Overview

This repository contains a multi-page portfolio site that highlights:
- frontend development skills
- backend development knowledge
- projects and featured work
- education and experience
- contact information

It also includes a small React/Vite project in the `my-project/` folder to demonstrate modern frontend development.

## Structure

- `frontend.html` — frontend skills and services
- `backend.html` — backend skills and services
- `projects.html` — project showcase
- `skills.html` — technical skill set
- `education.html` — educational background
- `contact.html` — contact form / details
- `header.html` / `footer.html` — reusable layout components
- `style.css` — shared styling
- `my-project/` — React + Vite demo application

## Features

- Clean portfolio layout
- Responsive navigation
- Separate pages for each portfolio section
- Modern React app example

## Technologies

- HTML
- CSS
- JavaScript
- React
- Vite
